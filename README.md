# UZISHOP

Este es el repositorio del proyecto UZISHOP.

## Descripción
Aplicación de comercio electrónico con interfaz moderna y funcionalidades completas.

## Estructura
- `/src`: Contiene el código fuente de la aplicación React.
- `/public`: Recursos estáticos.
- Archivos de configuración de Firebase (`firebase.json`, reglas, etc.).

## Instalación y Uso
1. Navega a la carpeta principal.
2. Instala las dependencias: `npm install`
3. Inicia el servidor de desarrollo: `npm run dev`

## Notas Importantes
Este proyecto utiliza variables de entorno para la configuración (ej. claves de API). Asegúrate de tener configurado tu archivo `.env` localmente.
